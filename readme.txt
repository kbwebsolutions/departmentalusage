 This report is designed to give an overview of a number of courses in an area.  You can choose the role to filter on and then in the report the user within that role.
 
 Install instructions.
 
 1. Download and unzip into /report folder
 
 2. Run notifications
 
 3. A new setting will ask you which role you want to filter on
 
 Go to Site Adminstration -> Reports -> Departmental Report to run the report
